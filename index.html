                    Key Features
                    
                

            



            

                
                
                
            

            

                

                    

                        
                        Server Connections
                    

                    
    
                        
  • 
                            
                            Connect to multiple Linux servers
                        
    • 
                        
  • 
                            
                            Support for Ploi and Laravel Forge
                        
    • 
                        
  • 
                            
                            Pause and resume tasks as needed
                        
    • 
                        
  • 
                            
                            Secure command execution
                        
    • 
                    

                    
Vanguard simplifies the process of connecting to your Linux servers for
                        backup operations. With support for multiple servers and various hosting providers, you can
                        efficiently manage your backup tasks from a single, localized dashboard.

                

                

                    

                        
                        Backup Solutions
                    

                    
    
                        
  • 
                            
                            MySQL and PostgreSQL database backups
                        
    • 
                        
  • 
                            
                            Regular file backups
                        
    • 
                        
  • 
                            
                            Support for various backup destinations
                        
    • 
                        
  • 
                            
                            Secure and efficient backup process
                        
    • 
                    

                    
Vanguard offers comprehensive data security and cloud backup solutions for
                        your critical information. With support for MySQL and PostgreSQL databases, as well as regular
                        file backups, you can ensure your critical data is safely stored and easily accessible when
                        needed.

                

                

                    

                        
                        Flexible Scheduling
                    

                    
    
                        
  • 
                            
                            Custom cron schedule support
                        
    • 
                        
  • 
                            
                            Pause and resume scheduled tasks
                        
    • 
                        
  • 
                            
                            Flexible task management
                        
    • 
                        
  • 
                            
                            Built on Laravel for reliability
                        
    • 
                    

                    
Vanguard's scheduling system gives you complete control over your backup
                        processes. Set up custom cron schedules for automated backups to fit your specific needs, and
                        manage your secure data storage efficiently. Built on Laravel, Vanguard ensures reliable and
                        timely execution of your scheduled tasks.
